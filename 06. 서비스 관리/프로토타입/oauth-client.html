OAuth 클라이언트 안내

  • Client ID / Secret: 그룹사 서비스가 마카다미아 멤버스 인증 API를 호출할 때 사용하는 인증 정보입니다.
  • Redirect URI: 로그인 완료 후 사용자를 되돌려 보낼 URL을 등록합니다. 보안을 위해 등록된 URI로만 리다이렉트됩니다.
  • Secret 재발급: Secret이 노출된 경우 상세 페이지에서 즉시 재발급받으세요. 기존 Secret은 즉시 무효화됩니다.
